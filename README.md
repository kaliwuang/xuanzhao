# 玄照 · XuanZhao

> 玄学为体，照见为用  
> **七术排盘 × 40+ 视角交叉印证 = 新一代群体智能预测系统**

玄照是一个玄学群体智能预测系统。输入出生信息，系统自动运行**八字、紫微斗数、奇门遁甲、大六壬、六爻纳甲、太乙神数、西洋占星**七术排盘，再喂给**诸葛亮、费曼、荣格、袁天罡、老子**等 40+ 个顶级认知框架从不同视角解读命盘，合成输出综合预测报告。

## 快速体验

```bash
# 克隆
git clone https://github.com/kaliwuang/xuanzhao.git
cd xuanzhao

# 安装依赖
pip install lunar-python pyswisseph

# 运行分析
python xuanzhao.py analyze --birth "2005-06-09 11:50" --location "呼和浩特" --gender male

# 生成 demo 报告
python xuanzhao.py demo --output my-report.md
```

**无需 Hermes Agent，纯 CLI 可跑。**

## 独立 CLI 工具

玄照提供完整的命令行工具，无需 Hermes Agent 即可运行：

```bash
# 命理分析（纯本地，无需网络）
python xuanzhao.py analyze --birth "2005-06-09 11:50" --location "呼和浩特" --gender male

# 导出 JSON 格式（便于程序处理）
python xuanzhao.py analyze --birth "2005-06-09 11:50" --location "呼和浩特" --gender male --format json

# 列出所有可用视角
python xuanzhao.py perspectives --list

# 生成示例报告
python xuanzhao.py demo --output my-report.md
```

### analyze 命令输出内容
- 八字四柱 + 纳音 + 藏干 + 十神
- 日主五行和旺衰
- 调候用神
- 大运序列
- 核心特征（子午冲、七杀透干等自动检测）
- 多视角解读框架

### predict 命令（需要 API Key）
```bash
python xuanzhao.py predict --birth "2005-06-09 11:50" \
  --location "呼和浩特" --gender male \
  --api-key "sk-..." --model "gpt-4o"
```

## Hermes Agent 集成

如需全自动群体智能预测，安装完整 Hermes skill：

```bash
bash install.sh
```

然后在 Hermes 中输入：

```
/load xuanzhao
预测我的事业走向，出生信息：2005-06-09 11:50，呼和浩特，男
```
