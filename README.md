# 玄照 · XuanZhao

> 玄学为体，照见为用  
> 东方命理 × 多视角群体智能 = 新一代预测系统

**玄照**是一个运行在 Hermes Agent 上的玄学群体智能预测系统。它融合了：

- **七术玄学**：八字、紫微斗数、奇门遁甲、大六壬、六爻纳甲、太乙神数、西洋占星
- **群体智能**：40+ 个顶级认知框架（诸葛亮、袁天罡、费曼、荣格、芒格、老子……）从不同视角独立解读命盘
- **交叉印证**：通过多视角共识/分歧分析，给出比单一命理师更立体的预测报告

## 系统要求

- Hermes Agent（推荐 v0.2+）
- Python 3.10+
- Node.js 18+（TokScale 用）
- Termux / Linux / macOS

## 快速安装

```bash
# 1. 克隆
git clone https://github.com/[your-username]/xuanzhao.git
cd xuanzhao

# 2. 安装依赖
pip install lunar-python sxtwl najia kinliuren kintaiyi iztro-py pyswisseph
npm install -g tokscale

# 3. 安装视角 skill
bash scripts/install-perspectives.sh

# 4. 复制 SKILL.md 到 Hermes
cp SKILL.md ~/.hermes/skills/xuanxue/xuanzhao/
```

## 使用示例

在 Hermes Agent 中输入：

```
/load xuanzhao
预测我的事业走向，出生信息：2005-06-09 11:50，呼和浩特，男
```

系统将自动：
1. 运行七术排盘
2. 选择 5 个最匹配的视角进行分析
3. 合成输出综合预测报告
