# 玄照 · XuanZhao — AI 玄学群体智能技能

> 玄学为体，照见为用。**无需 API Key，纯本地运行。**

## 概述

玄照是一个 **面向 AI Agent 的命理分析技能**。当用户提供出生信息时：

1. **八字排盘** — 基于 `lunar_python` 计算四柱、十神、纳音、藏干、空亡、大运
2. **视角引擎** — 108 个跨学科视角（含20个手写深度视角 + 76个模板视角 + 12个浅层视角），每个视角基于五行分布、十神力量、大运相位、冲刑害合等数据做结构化评分
3. **AI 解读** — **你（AI Agent）** 读取结构化数据并用你的能力进行综合解读。无需外部 LLM API。

**关键设计：不是让 xuanzhao 去调 API，而是让 AI Agent 成为解读引擎。**

## 安装

```bash
git clone https://github.com/kaliwuang/xuanzhao.git
cd xuanzhao
pip install lunar-python
```

## 使用工作流

### 第一步：命盘分析

当用户给出出生信息，运行分析获取结构化数据：

```bash
cd /path/to/xuanzhao
python xuanzhao.py analyze --birth "2005-06-09 11:50" --location "呼和浩特" --gender male --format json
```

输出 JSON 包含：
- `bazi` — 四柱、纳音、十神、藏干、空亡、大运
- `features` — 命局核心特征（子午冲、七杀透干等）
- `tiaohou` — 调候用神
- `perspectives` — 108 个视角的结构化评分
  - 每个视角有：`score`(0-100)、`dimensions`(多维评分)、`key_insights`、`warnings`、`advice`
- `astrology` — 西洋占星数据（如可用）

### 第二步：AI 自行解读

读取 JSON 输出后，**你作为 AI 需要做**：

1. **命盘总论** — 根据八字组合、日主强弱、五行分布给出核心判断
2. **视角共识分析** — 查看哪些视角评分较高/较低，找出共识模式
3. **综合解读** — 将结构化数据转化为有温度、有洞察的自然语言报告
4. **行动建议** — 基于大运相位和视角建议给出可操作的建议

**解读框架：**

```
报告结构建议：
1. 命盘总论（一句话概括核心特质）
2. 五行分析（什么元素过旺/过弱？意味着什么？）
3. 核心矛盾（基于features字段和视角disagreement）
4. 多视角共识（哪些视角对同一问题持相似看法？）
5. 大运展望（当前大运的影响和未来关键节点）
6. 综合建议（具体可操作的建议）
```

### 76个模板视角

覆盖中国哲学(10)、玄学命理(8)、西方哲学(15)、心理学(4)、科学创新(10)、商业领导力(12)、管理学家(5)、艺术家(5)、军事家政治家(8)。每个模板视角共享一套分析引擎，权重配比和解读文案各不相同。

## 重要注意事项

1. **所有分析基于数据计算，不是玄学迷信** — 透明可验证
2. **置信度说明** — 每个视角有 confidence 字段(0-1)，代表该视角在此命局上的适用性
3. **视角分歧是信息** — 不同视角评分差异大时，说明命局在不同维度上存在矛盾，这是最有价值的分析点
4. **大运动态** — 大运数据会随时间变化，建议在报告中注明"基于当前时间点"
5. **语言要求** — 用户是中文使用者，解读报告用中文书写

## 场景示例

### 用户说："帮我看看八字"

```
1. 问出生信息（时间、地点、性别）
2. python xuanzhao.py analyze --birth "..." --location "..." --gender 男 --format json
3. 读取 JSON 输出
4. 基于结构化数据进行综合解读
5. 输出自然语言报告
```

### 用户说："2005年6月9日11点50分呼和浩特出生的男的八字怎么样"

```
1. 直接运行分析
2. python xuanzhao.py analyze --birth "2005-06-09 11:50" --location "呼和浩特" --gender male --format json
3. 从输出中提取关键数据
4. 结合视角评分进行解读
5. 输出完整报告
```
