# 玄照 · XuanZhao — AI 玄学群体智能技能

> 玄学为体，照见为用。**无需 API Key，纯本地运行。**
> 知识来源：《八字理论精华宝典》OCR精读至P294/647，累计98个知识点。

## 概述

玄照是一个 **面向 AI Agent 的命理分析技能**。当用户提供出生信息时：

1. **八字排盘** — 基于 `lunar_python` 计算四柱、十神、纳音、藏干、空亡、大运
2. **视角引擎** — 108 个跨学科视角，基于五行分布、十神力量、大运相位、冲刑害合等数据做结构化评分
3. **AI 解读** — 你读取结构化数据并综合知识库进行解读

**关键设计：你的玄学知识 + 视角引擎的结构化数据 = 真正有深度的命理分析。**

**空亡互换核心断语：**
- 年柱与大运互换空亡 → 不利长亲（祖坟多空败）
- 月柱与大运互换空亡 → 女不利夫,男不利父母兄弟
- 日柱与大运互换空亡 → 主配偶不利
- 时柱与大运互换空亡 → 主子女不利
- 流年与大运互换空亡 → 一事无成

**空亡六大规则：**
① 遇六合不为空 ② 逢冲以吉论（纳音相生更好） ③ 不起旺衰平衡作用 ④ 大运空流年不空 ⑤ 同类型相冲极凶（如丁酉运丁卯年） ⑥ 逢六合不论空

## 使用工作流

### 第一步：命盘分析

```bash
python xuanzhao.py analyze --birth "2005-06-09 11:50" --location "呼和浩特" --gender male --format json
```

### 第二步：AI 解读

运行后读取JSON输出,按以下框架解读：

1. **命盘总论** — 八字组合、日主强弱、五行分布
2. **视角共识分析** — 哪些视角评分高/低,共识模式
3. **综合解读** — 将结构化数据 + 知识库转化为有温度的命理报告
4. **行动建议** — 大运相位和视角建议

**解读时如何调用知识库：**
- 分析十神时 → 参照"十神空亡"判断缺失
- 分析流年时 → 参照"神煞判词"给出当年吉凶
- 分析调候时 → 参照"调候用神表"判断用神是否到位
- 分析健康时 → 参照"主星疾病对应"和"疾厄宫三方四正"
- 下断语时 → 参照"铁口直断口诀"
