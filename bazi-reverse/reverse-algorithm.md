# 七术互逆 · 各入口倒推算法详解

## 总纲：统一搜索框架

所有入口共享一个核心算法：

```
1. 定义候选日期范围（年份范围 × 月份范围 × 日期范围 × 时辰范围）
2. 对每个候选 (y, m, d, h)：
   a. 用 lunar_python 算八字四柱
   b. 用 iztro-py 算紫微斗数
   c. 用 kinqimen 算奇门遁甲
   d. 用 najia 算六爻本命卦
   e. 用 kinliuren 算大六壬
   f. 用 kintaiyi 算太乙神数
   g. 用 pyswisseph 算西洋占星
3. 将生成的全七术数据与用户输入的特征做匹配
4. 输出匹配的候选日期
```

**范围优化**：年份范围越窄越好。默认 1900-2100 约 73,000 个日期 × 12 时辰 = ~876,000 候选。
每个候选生成全七术约需 0.01-0.1 秒，总时间不可接受。
必须用特征值**大幅缩小范围**。

## A. 八字入口（已实现）

### 特征值
四柱八字 = 年柱(2字) + 月柱(2字) + 日柱(2字) + 时柱(2字)

### 搜索优化
- 年柱每60年重复 → 1900-2100年只有3-4个候选年
- 月柱限制到1个节气月（~30天）
- 日柱每60天重复，30天内至多2个匹配
- 总候选：3年 × 2天 ≈ 6天

### 耗时
~5秒（脚本 reverse_bazi.py）

### 精度
✅ 极高。日柱唯一匹配，时柱验证一致。

## B. 紫微斗数入口

### 特征值 & 倒推价值

| 特征 | 描述 | 约束强度 |
|------|------|---------|
| 命宫地支 | 命宫在XX宫 | ★★★ 月+时确定 |
| 紫微星地支 | 紫微在XX宫 | ★★★ 日柱确定 |
| 五行局 | 金四局/火六局等 | ★★ 验证用 |
| 生年四化 | 某星化禄/权/科/忌 | ★★ 年干确定 |
| 主星分布 | 14主星位置 | ★★ 多星组合 |
| 大概年龄 | ~岁 | ★★ 解决60年周期 |
| 大限位置 | 当前大限在XX宫 | ★ 辅助 |

### 命宫 + 紫微星 联合约束

**命宫公式**：
- 命宫地支 = (寅月(2) + 月数 - 1 + 时支序号) % 12
- 即：月[1-12] + 时[0-11] → 命宫
  
**反推**：给定命宫在B宫 + 时支为C
- 月 = (B - C + 12 - 2 + 1) % 12 + 1
- 所以：命宫+时辰 → 唯一的月

**举例**：
- 命宫在午(6)，午时(时支=6，index=6)
- 月 = (6 - 6 + 12 - 2 + 1) % 12 + 1
- 月 = (0 + 12 - 1) % 12 + 1 = 12
- 所以是十二月

**紫微星公式**（简化）：
紫微星位置由 日干+日支 通过"紫微星诀"查表确定。
- 紫微在子 → 日柱必须在一组特定干支中
- 紫微在午 → 另一组特定干支
- 每六组日柱对应一个紫微位置

**联合约束**：
命宫+紫微星 → 月+日柱范围 → 通常锁定到3-7天/年

### 耗时估计
统一搜索 10年 × 12月 × 30天 × 12时辰 ≈ 43,200 次 gen+match
~30-60秒（主要取决于紫微排盘速度）

## C. 西洋占星入口

### 特征值 & 倒推价值

| 特征 | 描述 | 约束强度 |
|------|------|---------|
| 太阳星座 | 如"太阳双子" | ★★★ 锁定月份 |
| 上升星座 | 如"上升处女" | ★★★ 锁定时辰 |
| 月亮星座 | 如"月亮狮子" | ★★ 锁定日范围 |
| 水星/金星/火星 | 行星位置 | ★★ 辅助 |
| 宫位系统 | 行星在几宫 | ★ 辅助 |

### 搜索优化

1. **太阳星座 → 月份**：太阳每30天左右换座，精度±15天
2. **上升星座 → 时辰**：上升每2小时换一个星座，误差±1小时
3. **月亮星座 → 日期范围**：月亮每2-3天换座，给定月亮星座限到约3天窗口
4. **联合约束**：太阳月 × 月亮3天 × 上升2小时 ≈ 3-5个候选人/年

### 工具：pyswisseph

```python
import pyswisseph as swe
swe.set_ephe_path('/path/to/ephe')  # 需要星历文件

# 计算某时刻的太阳位置
jd = swe.julday(2005, 6, 9, 11.8333)  # 2005-06-09 11:50
sun_long = swe.calc(jd, swe.SUN)[0][0]
# 0-30°=白羊, 30-60°=金牛, ...
```

**注意**：Termux 环境可能需要星历文件。如果不可用，降级用近似公式估算太阳位置（经纬度到星座是±1天内精度）。

## D. 奇门遁甲入口

### 特征值 & 倒推价值

| 特征 | 描述 | 约束强度 |
|------|------|---------|
| 遁局 | 阳遁X局/阴遁Y局 | ★★★ 节气范围 |
| 值符 | 天XX星 | ★★ 时柱约束 |
| 值使 | XX门 | ★★ 时柱约束 |
| 时干/时支 | 当前时辰干支 | ★ 辅助 |
| 用神落宫 | 求测事项 | ★ 辅助 |

### 遁局约束

**节气和阴阳遁对应**：
- 阳遁：冬至 → 芒种（12月-6月）
- 阴遁：夏至 → 大雪（6月-12月）

**节气-局号映射**：
每节气（15天）对应3局，阳遁1-9，阴遁1-9。
- 冬至-小寒-大寒 → 阳遁1-3
- 立春-雨水-惊蛰 → 阳遁4-6
- ...
- 夏至-小暑-大暑 → 阴遁1-3
- ...

给定"阳遁三局" → 只能落在冬至到小寒到大寒三个节气（约45天窗口），且局号=3。

**值符天星 → 时柱**：
值符由时干决定（甲戊庚牛羊... 规则），可反推时柱范围。

**值使门 → 日柱**：
值使由日柱+时辰决定，反推日柱。

### 联合约束
遁局(45天) + 值符值使(日时) → 通常锁定到1-3天

### 工具：kinqimen

```python
from kinqimen import QimenPan
from lunar_python import Solar

s = Solar.fromYmdHms(2005, 6, 9, 11, 50, 0)
qm = QimenPan(s, 'yang')  # 或 'yin'
qm.dun  # 局数
qm.ju  # 完整局盘
```

## E. 六爻纳甲入口

### 特征值

| 特征 | 描述 | 约束强度 |
|------|------|---------|
| 本命卦 | 如"火风鼎" | ★★ 日柱约束 |
| 动爻 | 初九/九二等 | ★ 时柱约束 |
| 六亲分布 | 妻财子水、父母未土等 | ★ 辅助 |

### 精度限制

本命卦由日柱决定：每个日柱对应一个本命卦（纳甲装卦法）。
60天一周期，每个月最多2个匹配日。

**绝对不可单独用于倒推**，必须是：
- 本命卦 + 大概年份 → 2-6个候选
- 本命卦 + 年份 + 月份 → 1-2个候选

## F. 大六壬入口

### 特征值

| 特征 | 描述 | 约束强度 |
|------|------|---------|
| 课式 | 元首/重审/贼克等 | ★★ 日柱约束 |
| 三传 | 如"申戌子" | ★★ 日时约束 |
| 十二天将分布 | 青龙-天后-玄武 | ★ 辅助 |
| 四课 | 第一课至第四课 | ★★ 日时约束 |

### 精度限制

六壬课象由日柱+时辰决定。同一课式可能对应多个日期。
**需要年份+课式联合约束**。

### 工具：kinliuren

```python
from kinliuren import KR

kr.ju  # 课盘
```

## 总结：各入口实际可用性

| 入口 | 推荐程度 | 无需额外信息 | 需年龄 | 建议策略 |
|------|---------|------------|--------|---------|
| 八字四柱 | ⭐⭐⭐⭐⭐ | ✅ | 建议 | 直接倒推，准确 |
| 出生日期 | ⭐⭐⭐⭐⭐ | ✅ | 无需 | 直接排七术，最快 |
| 西洋占星 | ⭐⭐⭐⭐ | ⚠ 需年份 | 建议 | 太阳+上升+月亮联合 |
| 紫微斗数 | ⭐⭐⭐ | ⚠ 需年份 | 必要 | 命宫+紫微+生年干 |
| 奇门遁甲 | ⭐⭐⭐ | ⚠ 需年份 | 建议 | 遁局+值符+值使 |
| 大六壬 | ⭐⭐ | ❌ 需年份+月 | 必要 | 课式+三传 |
| 太乙神数 | ⭐⭐ | ❌ 需年份 | 必要 | 积年+主客算 |
| 六爻纳甲 | ⭐ | ❌ 需年份+月 | 必要 | 仅作辅助 |

**经验法则**：用户给的入口术数据越完整，倒推越精确。
如果倒推结果不唯一，让用户确认年龄段，或提供更多该术的特征细节。
