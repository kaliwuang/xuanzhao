# 玄照架构设计

## 三层架构

### 第一层：玄学排盘（七术）
每一术独立运行，输出结构化命盘数据。
各术关注点不同，形成互补。

### 第二层：多视角解读
40+认知框架分为四类：
- 中国玄学泰斗（命理/战略视角）
- 现代思想家（科技/商业视角）
- 西方哲学家（心理/逻辑视角）
- 西方神秘学（象征/自然视角）

### 第三层：合成输出
- **共识项**：2+视角指向同一结论
- **分歧项**：不同视角看法不同
- **置信度**：基于视角数量和一致性
- **盲区提醒**：所有视角可能忽略的维度

## 视角选择算法

根据问题类型自动匹配最佳视角组合。
也可手动指定关注的视角。
